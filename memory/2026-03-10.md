## 💡 重要决策

### 1. 配图系统完全 AI 化
**决策：** 所有提示词使用 AI 生成，不用模板

**理由：**
- 更专业、更准确
- 更有创意和细节
- 成本增加很小（约 1%），质量提升明显

### 2. 混合记忆模式
**决策：** 本地为主，云端为辅，优雅降级

**理由：**
- 本地完全可控、离线可用
- 云端作为补充和备份
- 不受单一服务影响

### 3. IP 监控增强
**决策：** 同时检测公共 IP 和微信 API

**理由：**
- VPN 分流可能导致 IP 误判
- 微信 API 直接验证更准确
- 反映真实可用情况

### 4. 自动推送 GitHub
**决策：** 每次更改后自动推送

**理由：**
- 系统化备份
- 版本管理
- 便于恢复和协作
