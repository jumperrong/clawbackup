## 🔧 使用方法

### 方法 1：独立优化

```bash
# 优化单个提示词
python3 scripts/optimize_prompt.py \
  --prompt "健康科普插图，主题：臀肌挛缩康复" \
  --title "什么是臀肌挛缩？" \
  --style "干货"
```

### 方法 2：集成到配图

```bash
# 为文章配图（使用 AI 优化）
python3 scripts/add_article_images.py \
  --article "scripts/output/articles/article_xxx.md" \
  --topic "臀肌挛缩康复指南" \
  --style "干货" \
  --optimize  # 启用 AI 优化
```
