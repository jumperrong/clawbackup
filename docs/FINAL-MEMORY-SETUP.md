## 🔄 工作流程

### 记忆保存

```
1. 对话结束
   ↓
2. 保存到本地 MEMORY.md
   ↓
3. 尝试保存到 MemOS Cloud
   ├─ 成功 → ✅ 云端备份完成
   └─ 失败 → ⚠️  仅本地保存（不影响使用）
```

### 记忆检索

```
1. 用户查询
   ↓
2. 本地 FAISS 索引检索（<100ms）
   ↓
3. （可选）尝试 MemOS Cloud 检索
   ├─ 成功 → 合并结果
   └─ 失败 → 仅使用本地结果
   ↓
4. 返回结果
```
