# 交互式写作模式使用指南

_分步生成、逐步审核、完美控制文章质量_

---

## 🎯 什么是交互式写作？

交互式写作模式是一个**分步引导**的创作流程，让你：

- ✅ **逐步生成** - 大纲 → 正文 → 配图，每一步都可控制
- ✅ **实时审核** - 每步完成后都可以修改和调整
- ✅ **多轮对话** - 随时提出修改意见，AI 即时调整
- ✅ **完美控制** - 不满意就重来，直到满意为止

## 📋 完整流程

### 步骤 1：确定主题和标题 ⏱️ 约 2 分钟

**AI 会问：**
```
请告诉我：
1. 你想写什么主题的文章？
2. 目标读者是谁？
3. 文章字数大概多少？
```

**你回答：**
```
👉 主题：康复训练
👉 标题：臀肌挛缩不用开刀？这份康复指南火了
```

**AI 确认：**
```
✅ 已确认:
   主题：康复训练
   标题：臀肌挛缩不用开刀？这份康复指南火了
```

**你可以：**
- ✅ 确认进入下一步
- ✅ 重新修改主题/标题

---

3. 核心内容：详细讲解主要知识点
4. 案例分析：通过实际案例说明
5. 总结：回顾要点，给出建议

## 💡 实用技巧

### 技巧 1：中途修改

在任何一个步骤，如果突然想修改前面的内容：

```bash
# 在生成正文时，想修改标题
👉 [Ctrl+C] 中断当前流程
👉 python3 ai_assistant.py "交互式写作"
# 重新开始，这次输入正确的标题
```

### 技巧 2：保存进度

交互式写作会自动保存进度：

```bash
# 保存位置
scripts/output/articles/article_YYYYMMDD_HHMMSS.md

# 下次可以继续编辑
python3 export_wechat.py --input article_20260311_131500.md
```

### 技巧 3：批量生成

如果要生成多篇文章：

```bash
# 第一篇
python3 interactive_writer.py

# 完成后，继续第二篇
python3 interactive_writer.py
```

## 🎯 使用场景

### 场景 1：需要精细控制的文章

**适合：** 专业科普、技术文章、医疗内容

**原因：** 每一步都可以审核，确保准确性

**流程：**
1. 仔细审核大纲结构
2. 逐步调整正文内容
3. 生成专业配图
4. 选择合适主题

---

### 场景 2：不确定方向的文章

**适合：** 新主题、新领域、首次尝试

**原因：** 可以边写边调整，不怕走偏

**流程：**
1. 先确定大致方向
2. 生成大纲看看是否合理
3. 不满意就调整大纲
4. 确认后再开始写正文

---

### 场景 3：需要配图的文章

**适合：** 教程类、产品评测、案例分析

**原因：** 配图提示词可以逐个审核和调整

**流程：**
1. 生成正文后，AI 自动分析需要配图的位置
2. 为每个位置生成提示词
3. 逐个查看和调整
4. 确认后再去生成图片

## ❓ 常见问题

### Q: 交互式写作和普通模式有什么区别？

**A:** 

| 特性 | 普通模式 | 交互模式 |
|------|---------|---------|
| 流程 | 一键生成 | 分步引导 |
| 控制 | 生成后修改 | 每步都可修改 |
| 审核 | 最后审核 | 逐步审核 |
| 时间 | 快（1-2 分钟） | 慢（15-20 分钟） |
| 质量 | 依赖 AI | 人工 + AI |
| 适合 | 简单文章 | 重要文章 |

---

### Q: 交互式写作需要多长时间？

**A:** 通常 15-20 分钟

- 步骤 1（主题）：2 分钟
- 步骤 2（大纲）：3 分钟
- 步骤 3（正文）：5-10 分钟
- 步骤 4（配图）：3 分钟
- 步骤 5（主题）：2 分钟
- 步骤 6（导出）：2 分钟

---

### Q: 可以在中途退出吗？

**A:** 可以！按 `Ctrl+C` 退出，已生成的内容会自动保存。

---

### Q: 修改大纲后，正文会自动更新吗？

**A:** 目前不会。需要重新生成正文。

**未来计划：** 智能识别修改，只更新相关章节。

---

### Q: 配图提示词如何使用？

**A:** 

1. 复制提示词
2. 打开 AI 绘画工具（豆包、Midjourney 等）
3. 粘贴提示词
4. 生成图片
5. 下载并用于文章

---

## 🎉 总结

**交互式写作模式适合：**

- ✅ 需要精细控制的专业文章
- ✅ 不确定方向的探索性写作
- ✅ 需要配图的教程类文章
- ✅ 对质量要求高的重要文章

**不适合：**

- ❌ 简单的日常分享（用普通模式更快）
- ❌ 时间紧迫的文章（普通模式 1-2 分钟）
- ❌ 已有完整草稿的文章（直接导出即可）
