# 模型选择指南

## 可用模型

| 模型 ID | 说明 | 上下文 | 最大输出 | 适用场景 |
|---------|------|--------|---------|---------|
| `qwen3.5-plus` | 千问最新一代，支持文本 + 图像 | 1M | 65K | **默认通用** - 日常对话、信息查询、多模态 |
| `qwen3-max-2026-01-23` | 千问 3 旗舰模型 | 262K | 65K | 复杂推理、高质量回答 |
| `qwen3-coder-next` | 代码专用模型 | 262K | 65K | 快速代码任务 |
| `qwen3-coder-plus` | 代码专用模型（推荐） | 1M | 65K | **编程任务首选** - 代码编写、审查、重构 |
| `MiniMax-M2.5` | MiniMax 模型 | 1M | 65K | 长文本处理、创意写作 |
| `glm-5` | 智谱 GLM-5 | 202K | 16K | 中文任务、知识问答 |
| `glm-4.7` | 智谱 GLM-4.7 | 202K | 16K | 轻量中文任务 |
| `kimi-k2.5` | 月之暗面 Kimi，支持图像 | 262K | 32K | 长文档阅读、图像理解 |

## 选择策略

### 按任务类型

```
日常对话/生活问题     → qwen3.5-plus (默认)
编程/代码任务         → qwen3-coder-plus
复杂推理/分析         → qwen3-max-2026-01-23
长文档/长文本         → qwen3.5-plus 或 MiniMax-M2.5 (1M 上下文)
图像理解             → qwen3.5-plus 或 kimi-k2.5
中文知识问答         → glm-5
```

### 按场景

- **飞书聊天** - qwen3.5-plus (平衡速度和能力)
- **代码审查** - qwen3-coder-plus
- **技术方案设计** - qwen3-max-2026-01-23
- **读取长文件** - qwen3.5-plus (1M 上下文)
- **图像分析** - qwen3.5-plus

## 切换模型

在当前会话中临时切换：
```
/model qwen3-coder-plus 帮我写个函数
```

或在 openclaw.json 中修改默认：
```json
"agents": {
  "defaults": {
    "model": {
      "primary": "bailian/qwen3-coder-plus"
    }
  }
}
```
